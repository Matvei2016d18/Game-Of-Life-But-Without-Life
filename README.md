#How to run the game?
To run the game, you need to select a version,
download the jar file from the repository,
and run it in Java. If you have access to
Java 8, you can download the jar file that was
created using this version.

#How to create your own version?
I recommend using the Classic
version, downloading the java file, making
your changes, and running the compiler. I
recommend using JDK 8!
