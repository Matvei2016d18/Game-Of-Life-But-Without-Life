# How to run the game?
To run the game, you need to select a version, download the `.jar` file from the repository, and run it in **Java**.  
If you have access to **Java 8**, you can download the `.jar` file that was created using this version.

# How to create your own version?
I recommend using the **Classic version**: download the `.java` file, make your changes, and run the compiler.  
I recommend using **JDK 8**!
